🚀 AI Project – Medical Image Analysis (Deep Learning)
an exciting project focused on applying Artificial Intelligence to healthcare, specifically in wireless capsule endoscopy image analysis.
🔍 The objective was to assist medical practitioners by automatically detecting abnormalities in gastrointestinal images.
💡 What I built:
A deep learning pipeline using VGG16, VGG19, and Xception models
Automatic classification of images into normal vs pathological cases
Performance comparison, where Xception achieved the best results
🖥️ In addition, I developed a desktop application (C# + Python integration) that allows:
Image visualization (slideshow, navigation)
Image processing (zoom, rotation, brightness/contrast)
Advanced analysis (edge detection: Sobel, Laplacian, Canny)
🎯 The goal is to reduce diagnosis time and support decision-making for medical professionals.

✔️ Deep Learning (TensorFlow / Keras)
✔️ Computer Vision
✔️ Model evaluation (Accuracy, Precision, Recall)
✔️ Software development and AI integration

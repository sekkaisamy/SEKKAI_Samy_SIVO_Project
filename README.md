  <h1>🚀 AI Project – Medical Image Analysis (Deep Learning)</h1>

  <p>
    An exciting project focused on applying Artificial Intelligence to healthcare,
    specifically in wireless capsule endoscopy image analysis.
  </p>

  <p>
    🔍 The objective was to assist medical practitioners by automatically detecting
    abnormalities in gastrointestinal images.
  </p>

  <h2>💡 What I built:</h2>
  <ul>
    <li>A deep learning pipeline using VGG16, VGG19, and Xception models</li>
    <li>Automatic classification of images into normal vs pathological cases</li>
    <li>Performance comparison, where Xception achieved the best results</li>
  </ul>

  <h2>🖥️ In addition, I developed a desktop application (C# + Python integration) that allows:</h2>
  <ul>
    <li>Image visualization (slideshow, navigation)</li>
    <li>Image processing (zoom, rotation, brightness/contrast)</li>
    <li>Advanced analysis (edge detection: Sobel, Laplacian, Canny)</li>
  </ul>

  <p>
    🎯 The goal is to reduce diagnosis time and support decision-making for medical professionals.
  </p>

  <h2>This project strengthened my skills in:</h2>
  <ul>
    <li>✔️ Deep Learning (TensorFlow / Keras)</li>
    <li>✔️ Computer Vision</li>
    <li>✔️ Model evaluation (Accuracy, Precision, Recall)</li>
    <li>✔️ Software development and AI integration</li>
  </ul>
